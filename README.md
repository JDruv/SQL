# 🗄️ Raccolta Script SQL: Da Base ad Avanzato

Benvenuto in questo repository! Qui troverai una collezione di script SQL pensati per esplorare e mettere in pratica il linguaggio SQL, partendo dai concetti più semplici fino ad arrivare a tecniche più complesse e avanzate.

## 📖 Descrizione del Progetto

Questo progetto è una libreria di riferimento per chiunque voglia imparare, ripassare o approfondire SQL. I file sono divisi per livello di difficoltà e argomento, e includono query di estrazione dati, manipolazione, funzioni e procedure.

## 📂 Struttura del Repository

Il repository è organizzato in diverse categorie per facilitare la navigazione:

### 🟢 1. Livello Base (`/01-base`)
* **Query Fondamentali:** `SELECT`, `WHERE`, `ORDER BY`, `LIMIT`.
* **Manipolazione Dati (DML):** `INSERT`, `UPDATE`, `DELETE`.
* **Definizione Dati (DDL):** Creazione di tabelle (`CREATE`), modifica (`ALTER`) ed eliminazione (`DROP`).
* **Join Semplici:** `INNER JOIN`, `LEFT JOIN`, `RIGHT JOIN`.

### 🟡 2. Livello Intermedio (`/02-intermedio`)
* **Aggregazioni:** `GROUP BY`, `HAVING`, funzioni come `COUNT()`, `SUM()`, `AVG()`.
* **Subquery:** Query annidate e clausole `IN`, `EXISTS`, `ANY`, `ALL`.
* **Manipolazione Stringhe e Date:** Funzioni predefinite per la gestione di testi e formati temporali.
* **Viste (Views):** Creazione e utilizzo di viste per semplificare query complesse.

### 🔴 3. Livello Avanzato (`/03-avanzato`)
* **Window Functions:** `ROW_NUMBER()`, `RANK()`, `LEAD()`, `LAG()`.
* **CTE (Common Table Expressions):** Clausola `WITH` per query ricorsive e leggibili.
* **Stored Procedures & Funzioni:** Creazione di funzioni personalizzate e procedure con parametri.
* **Trigger:** Automazione di operazioni sul database ad eventi specifici (es. `AFTER INSERT`).
* **Gestione Transazioni:** `COMMIT`, `ROLLBACK` e gestione degli errori.
